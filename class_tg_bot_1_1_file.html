Detailed Description

This object represents a file ready to be downloaded. The file can be downloaded via Api::downloadFile or via the link https://api.telegram.org/file/bot<token>/<filePath>. It is guaranteed that the File::filePath will be valid for at least 1 hour. When the File::filePath expires, a new one can be requested by calling Api::getFile.

The maximum file size to download is 20 MB
